# 📝 To Do App (Do Mo)

A simple and clean To Do App built with React.

## Features

* Add new tasks
* Remove completed tasks
* Instant updates with React State
* Simple and user-friendly design

## Built With

* React.js
* JavaScript
* CSS

## Purpose
This project was created to practice React fundamentals such as:

* Components
* useState Hook
* Event Handling
* List Rendering
